Branching and Collaboration

  • Create a branch:
    git checkout -b feature1
  • Merge branch to main:
    git merge feature1
  • Resolve merge conflicts (guided):
    1. Open conflicting files in your code editor.
    2. Look for conflict markers: <<<<<<<, =======, >>>>>>>
    3. Decide which changes to keep and remove the markers.
    4. Save the file and run: git add <filename>
    5. Complete the merge with: git commit
