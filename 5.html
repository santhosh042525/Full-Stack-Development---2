Upload React Project to GitHub

  1. Create a new React app:
    npx create-react-app myapp
  2. Navigate into project folder:
    cd myapp
  3. Initialize Git repository:
    git init
  4. Add files and commit:
    git add .
    git commit -m "Initial commit"
  5. Connect to GitHub remote:
    git remote add origin https://github.com/username/myapp.git
  6. Push to GitHub:
    git push -u origin main
  7. .gitignore already excludes node_modules by default in React apps.
  8. Create feature branches:
    git checkout -b feature/navbar
    git checkout -b feature/form
  9. Merge branches to main:
    git checkout main
    git merge feature/navbar
    git merge feature/form
  10. Practice pull requests:
    Use GitHub GUI to create pull requests, review changes, and merge.
